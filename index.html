Quem somos?

O RuntimeProgramInterfaceCheck(RPIC) é um projeto feito para inspecionar o funcionamento de máquinas, desenvolvido por Davi Andrade Macedo, Isabela Silvestre Rodrigues, Pedro Augusto Lopes Ramos de Oliveira e Renato Oliveira dos Santos.

Qual a função?

O RPIC é um software que executa funções embutidas do sistema operacional dos computadores para realizar a checagem pela integridade do próprio sistema operacional, seus utilitários e software.
