Bem-vindo ao RPIC!

Quem somos?

O RuntimeProgramInterfaceCheck(RPIC) é um projeto feito para inspecionar o funcionamento de máquinas, desenvolvido por Wannakry, Silver, Ame e ED.

Qual a função?

O RPIC é um software que executa funções embutidas do sistema operacional dos computadores para realizar a checagem pela integridade do próprio sistema operacional, seus utilitários e software.

A quem pode interessar?

Este projeto pode ser de interesse de:

  1. Empresas
  2. Escritórios
  3. Técnicos em Informática
  4. Administradores de Sistema
  5. Usuários comuns de computador
